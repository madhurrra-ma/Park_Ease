#  ParkEase

> **Smart Parking Solution – Frontend Mobile Application**

ParkEase is a smart parking solution designed to reduce the time spent searching for parking spaces by combining **IoT-enabled parking sensors**, **cloud processing**, and a **React Native mobile application**. The overall system provides users with real-time parking availability, interactive maps, parking layouts, and navigation to nearby parking locations.

> **Note:** This repository contains the **frontend mobile application** developed as part of the ParkEase team project.

---

## About the Project

Urban drivers spend a significant amount of time searching for parking, leading to traffic congestion, fuel wastage, and inefficient use of available parking spaces. ParkEase addresses this challenge by providing users with live parking information through an intuitive mobile application connected to an IoT-based smart parking infrastructure.

---

## Frontend Features

*  Search nearby parking locations
*  Interactive map displaying parking locations
*  Filter parking by type (Paid / Free / Reserved)
*  View parking details
*  Visual parking slot layout
*  Navigate to parking locations using Google Maps
*  Responsive mobile interface built with React Native

---

##  Overall System Architecture

The complete ParkEase solution consists of:

* ESP32-based parking slot monitoring
* Ultrasonic sensors and magnetometers for vehicle detection
* BLE communication between sensor nodes
* Cloud-based processing of parking status
* React Native mobile application for users

**This repository includes only the React Native frontend application.** The IoT firmware and cloud backend are part of the complete team project and are not included in this repository.
